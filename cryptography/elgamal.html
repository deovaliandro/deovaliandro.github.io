El-Gamal

Properties

  • n, a prime number (not secret)
  • g, primitive root of n (not secret)
  • x, a random number
  • y = g^x mod p
  • m, message
  • a and b, pair of ciphertext

Result

Alice's key: private key = x, publick key = (y, g, p). Ciphertext = (a, b).

usage: python elgamal.py
